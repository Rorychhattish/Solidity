# 🗳️ Voting Smart Contract

A Solidity smart contract that implements a simple decentralized voting system with proposal creation, member-based voting, vote updates, and automatic proposal execution.

## ✨ Features

* Member authorization
* Proposal creation
* Yes / No voting
* Vote updating
* Automatic proposal execution
* Event emission
* Access control using modifiers

## 🛠️ Built With

* Solidity ^0.8.20
* Remix IDE
* Ethereum
